# CF-PROBE

CF-PROBE 是一个基于 Cloudflare Worker 的探针前端修改版。

本项目基于 [CF-Server-Monitor-Pro](https://github.com/a63414262/CF-Server-Monitor-Pro) 二次修改而来，主要用于展示服务器 / VPS 探针信息，并在原项目基础上增加了全球点亮、全屏壁纸开关等自定义前端效果。

## 项目来源

本项目基于以下原项目二次修改而来：

- 原项目名称：CF-Server-Monitor-Pro
- 原项目作者：a63414262
- 原项目地址：https://github.com/a63414262/CF-Server-Monitor-Pro
- 原项目授权：原项目 README 中标注为 MIT License

感谢原作者的开源贡献。本仓库是在原项目基础上的个人修改版本，并非原作者官方版本。

## 修改内容

相比原项目，本版本主要进行了以下修改：

1. 增加全球点亮功能
2. 增加全屏壁纸开关功能
3. 调整部分前端显示效果
4. 对页面样式和展示逻辑进行了个人化修改
5. 根据个人使用需求对部分代码进行了整理和调整
